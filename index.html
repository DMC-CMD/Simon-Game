press a key to start!

To start the game press a key (not important which one)
A colour pattern plops up.
You need to remember it.
Now click in the right order on the buttons.
If the color pattern is right you get to the next level.
If you make a mistake you can restart the game by simply pressing a key.
In Level 1 there is only one color to remember,
but with every increasing level, another color is added to the pattern.
